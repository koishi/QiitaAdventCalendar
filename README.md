# QiitaAdventCalendar

Qiita Advent Calendar Viewer for iOS

http://qiita.com/advent-calendar/

## Installation

$ git clone https://github.com/koishi/QiitaAdventCalendar<br>
$ bundle install<br>
$ bundle exec pod install

## Usage

Run iOS Simulator or iOS Devices<BR>iOS 9.0 and Later
